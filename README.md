# 💫 About Me:
I'm a data and ML engineer with 4+ years of experience building and shipping end-to-end AI systems in production, actual deployed systems that creates impact.<br>

My background spans the full ML stack. At the Regional Transportation Commission of Nevada, I independently designed and productionized a YOLO-based computer vision pipeline for real-time ridership counting built the annotation pipeline from scratch on 5,000+ labeled frames, fine-tuned the model using precision-recall evaluation, and deployed it in a containerized Python/Docker environment at 90% detection accuracy. Around the same time I shipped a generative AI Text-to-Speech application that eliminated 100% of manual audio generation for the transit fleet that one went from problem identification to production without any external engineering support.<br>

On the LLM side, I've built RAG systems using LangChain, FAISS vector search, and open-weight models  Mistral, Llama, Gemma  learning firsthand that retrieval engineering matters more than model selection in applied AI systems. I'm also currently building a full photogrammetry-to-3D reconstruction pipeline using COLMAP, Gaussian Splatting, and Three.js purely because the problem of converting real-world video into georeferenced 3D representations is one of the most interesting perception challenges I've encountered.What makes me different from most ML practitioners is that I don't just train models I own outcomes.<br>

I'm drawn to roles where that full-ownership mindset is valued where the expectation is to move fast, ship real systems, and compound learning into better and better solutions over time.
